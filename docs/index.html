Microsite

This is a microsite initialized from chadoh/microsite-template.

The entire production site is one minified file with few (if any) JS dependencies, but the development version uses nice tooling and multiple files.

This is great for making explainer websites or other simple sites.

These sites can also easily be hosted for free on GitHub. For more info, see the README files in the root and the docs folder, as well as scripts in package.json.
